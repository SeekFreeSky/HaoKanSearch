📖 简介
在浏览 好看视频 时，经常会发现一些有趣的搬运视频，但想去源平台（如抖音、B站）看高清原版或看评论？

HaoKanSearch 在页面悬浮一个胶囊式工具栏，自动提取当前视频的纯净标题，一键跳转到 抖音 或 Bilibili 搜索同款视频。

✨ 核心特性
1. 🎯 精准分流搜索
独立按钮：告别“点击一下弹两个窗”的烦恼。脚本提供独立的 [抖音] 和 [B站] 按钮，想搜哪里点哪里。

智能清洗：自动去除标题中的 🔥、😂 等 Emoji 表情、#话题、【高清】 等无关后缀，确保搜索关键词最精准。

2. 🖐️ 极致交互体验
胶囊设计：采用半透明磨砂黑底 + 品牌色点缀（抖音红/B站蓝），UI 现代且不遮挡视线。

随意拖拽：按住胶囊空白处即可拖动到屏幕任意位置。

位置记忆：刷新页面或下次打开，按钮依然在你熟悉的地方。

触屏支持：完美支持 iPad、安卓平板及手机浏览器的手指拖拽。

3. ⚡ 极客级功能
跨标签页同步：你在 A 标签页移动了按钮，B 标签页的按钮会自动同步位置，多开党福音。

全屏自动隐藏：看视频全屏时自动隐身，退出全屏自动出现，零视觉干扰。

右键复制：在按钮上点击鼠标右键，即可将清洗后的纯净标题复制到剪贴板。

SPA 无缝支持：完美适配好看视频的单页应用机制，切换视频无需刷新，按钮自动更新状态。

🎮 使用说明
操作	效果
左键点击 "抖音"	新标签页打开抖音搜索结果
左键点击 "B站"	新标签页打开 Bilibili 搜索结果
按住拖动	移动悬浮窗位置 (自动保存)
右键点击	复制当前视频标题
